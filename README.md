## 📌 Project Overview

This project focuses on **multi-class classification of dry beans** using machine learning techniques.
It includes complete data analysis, preprocessing, model building, and deployment using Streamlit.

## 📊 Dataset

* Dataset: Dry Bean Dataset
* Contains multiple bean types (classes)
* Features include shape, size, and texture measurements

## 🔍 Exploratory Data Analysis (EDA)

* Class distribution visualization
* Histogram plots for feature distribution
* Boxplots for outlier detection
* Missing value & duplicate handling

---

## ⚙️ Data Preprocessing

* Removed duplicate values
* Outlier treatment using **IQR method**
* Train-test split with stratification
* Feature scaling
* Handling skewness using log transformation
* Smote To Handle Class Imbalance
* HyperParameter Tuning(Randomize Search cv)

---

## 🤖 Machine Learning Models Used

* Logistic Regression (Softmax)
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest
* Gradient Boosting
* CatBoost Classifier
* CatBosst After HyperParameter Tining
